# 标尺

画布顶部和左边标尺栏

![乐吾乐meta2d标尺](/img/rule.png)

在线体验： http://t.le5le.com/

示例：

```js
// 设置默认缺省标尺属性
meta2d.store.options.rule = true; // 开启
meta2d.store.options.ruleColor = 'eeeeee'; // 标尺颜色， 可缺省

// 设置单个图纸的标尺属性
meta2d.store.data.rule = true; // 开启
meta2d.store.data.ruleColor = 'eeeeee'; // 标尺颜色， 可缺省
```
